本网站使用[VuePress Theme Hope](https://theme-hope.vuejs.press/zh/)主题进行搭建

### 1、搭建VuePress 的运行环境
[下载地址](https://nodejs.org/en)

```bash
node >v18.12.0
```
```bash
#nvm包管理器切换node版本
nvm use 20.9.0
```


### 2、开启corepack
在你安装 Node.js 之后，在终端中输入下列命令启用 corepack: pnpm
```bash
corepack enable
```

### 3、安装node_moudle模块
```bash
npm install
```

### 4、启动项目
```bash
pnpm docs:dev
```
